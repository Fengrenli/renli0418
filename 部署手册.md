# 项目部署手册

## 1. 环境要求

- Node.js 版本：18.18.0（本地/线上一致）
- 依赖安装：npm install（本地）/ npm install --production（线上）

## 2. 一键部署

- Linux/macOS：./deploy.sh
- Windows：./deploy.ps1

## 3. 常用命令

- 查看后端日志：pm2 logs renliyesheng
- 重启后端服务：pm2 restart renliyesheng
- 备份线上代码：cp -r /www/wwwroot/renliyesheng /www/wwwroot/renliyesheng_backup_$(date +%Y%m%d)

## 4. 问题排查

- 接口连不上：检查 PQS 白名单 + 后端 PM2 日志
- 前端样式错乱：确认前端打包时用了生产环境配置（.env.production）
- 登录跳转错误：检查前端路由配置 + 后端 token 逻辑

## 5. 环境变量配置

### 前端环境变量
- .env.development：本地开发环境
- .env.production：线上生产环境

### 后端环境变量
- .env：线上服务器环境配置

## 6. 部署流程

1. 本地修改代码
2. 执行部署脚本
3. 查看部署日志确认成功
4. 访问线上项目验证修改生效

## 7. 备份策略

- 数据库：阿里云 PQS 自动备份
- 代码：服务器定期备份 + Git 版本控制
