# 成都仁力烨升国际贸易有限公司 - 功能开发PRD

## 文档信息
- **文档名称**：功能开发PRD
- **版本**：v1.0
- **创建日期**：2026年3月23日
- **适用范围**：品牌客户专属登录、物流实时追踪API、在线即时通讯功能开发

## 一、项目背景

成都仁力烨升国际贸易有限公司专注于海外店面建设、BIM设计及综合供应链解决方案，已成功助力小龙坎、吼堂、大龙燚、杨国福等知名品牌在全球100+核心城市落地。为了提升客户体验和业务效率，现计划开发以下功能：

1. **品牌客户专属登录入口**：为每个大品牌客户提供专属账号，实现数据隔离
2. **物流实时追踪API**：接入国际货运API，实现物流信息自动同步
3. **在线即时通讯**：增加WhatsApp或微信客服悬浮按钮，提升客户沟通效率

## 二、功能需求

### 1. 品牌客户专属登录入口

#### 1.1 功能描述
- 为每个品牌客户（如小龙坎、吼堂等）创建专属登录账号
- 登录后只能查看和管理自己名下的项目
- 提供项目进度查看、BOM历史查询等功能
- 数据严格隔离，确保品牌间信息不泄露

#### 1.2 技术实现
- **用户角色扩展**：在现有用户表基础上增加`brand_id`字段，标识品牌归属
- **权限控制**：实现基于品牌的RBAC权限控制
- **前端页面**：创建品牌客户专用的登录页面和仪表盘
- **API接口**：修改现有API，增加品牌过滤逻辑

#### 1.3 页面设计
- **登录页面**：品牌客户专用登录界面，包含品牌logo
- **仪表盘**：显示品牌专属项目列表和关键指标
- **项目详情**：仅显示品牌相关项目的详细信息
- **BOM管理**：仅显示品牌相关的BOM项目

### 2. 物流实时追踪API

#### 2.1 功能描述
- 接入主流国际货运公司API（如DHL、顺丰国际等）
- 自动同步物流信息到系统
- 为每个物流订单创建追踪记录
- 在项目页面显示实时物流状态

#### 2.2 技术实现
- **API集成**：开发DHL、顺丰国际等API客户端
- **物流追踪服务**：创建定时任务，定期同步物流信息
- **数据模型**：扩展现有项目表，增加物流追踪字段
- **前端展示**：在项目详情页面增加物流追踪模块

#### 2.3 数据源
- **DHL API**：https://developer.dhl.com/
- **顺丰国际API**：https://www.sf-express.com/cn/sc/dynamic_function/api/
- **其他货运公司API**：根据业务需求扩展

### 3. 在线即时通讯

#### 3.1 功能描述
- 网站右下角添加WhatsApp和微信客服悬浮按钮
- 点击按钮直接跳转到对应通讯软件
- 支持预设消息模板，方便客户快速咨询
- 后台可配置客服账号和自动回复内容

#### 3.2 技术实现
- **前端组件**：创建悬浮按钮组件
- **配置管理**：在后台添加客服账号配置功能
- **消息模板**：支持自定义消息模板
- **多语言支持**：根据用户语言显示对应联系方式

#### 3.3 集成方式
- **WhatsApp**：使用whatsapp://send URL scheme
- **微信**：使用微信二维码或小程序链接
- **其他通讯工具**：根据业务需求扩展

## 三、技术架构

### 1. 前端技术
- **框架**：React + TypeScript
- **状态管理**：Context API + useReducer
- **UI组件**：自定义组件 + Tailwind CSS
- **路由**：React Router

### 2. 后端技术
- **语言**：Node.js + TypeScript
- **框架**：Express
- **数据库**：PostgreSQL
- **缓存**：Redis（可选）
- **API集成**：axios + 自定义API客户端

## 五、实施风险

### 1. 技术风险
- **API集成**：不同货运公司API格式差异大，可能需要适配
- **数据同步**：物流信息实时性要求高，需要合理设计同步策略
- **权限控制**：品牌数据隔离需要严格的权限验证

### 2. 业务风险
- **品牌接受度**：需要与品牌客户沟通，确保功能符合需求
- **API成本**：部分货运API可能收取使用费用
- **数据安全**：品牌数据需要额外的安全措施

### 3. 应对策略
- **API适配**：开发统一的API适配器，降低集成复杂度
- **数据同步**：采用增量同步和缓存策略，提高性能
- **权限控制**：实施多层权限验证，确保数据安全
- **成本控制**：评估API使用成本，选择性价比高的方案
