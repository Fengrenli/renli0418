# AI功能降级处理说明

## 项目中的AI功能

本项目使用了谷歌GenAI API提供以下AI功能：

1. **BOM生成器中的HS编码建议** - 自动为材料生成HS编码
2. **项目智能创建** - 通过自然语言描述自动生成项目信息和地理坐标
3. **图片分析** - 通过图片识别材料类型并生成HS编码

## 降级处理机制

由于谷歌AI服务在某些网络环境下可能无法访问，我们实现了以下降级处理机制：

### 1. BOM生成器
- **AI不可用时**：系统会显示友好的提示信息，建议用户手动输入HS编码
- **手动替代方案**：用户可以直接在HS编码输入框中手动输入编码
- **错误处理**：当AI服务调用失败时，系统会捕获错误并显示相应的提示信息，确保用户体验不受影响

### 2. 项目创建
- **AI不可用时**：系统会提示用户使用手动添加项目的方式
- **手动替代方案**：用户可以通过"手动添加项目"功能创建新项目，填写所有必要的信息
- **错误处理**：当AI服务调用失败时，系统会显示提示信息，引导用户使用手动方式

### 3. 管理员门户
- **AI不可用时**：系统会提示管理员通过项目列表手动添加项目
- **手动替代方案**：管理员可以在项目列表中直接添加新项目，填写详细信息
- **错误处理**：当AI服务调用失败时，系统会显示提示信息，确保管理员可以继续使用核心功能

## 部署到阿里云的注意事项

1. **网络配置**：确保服务器能够访问必要的外部服务
2. **API密钥**：在`.env`文件中配置有效的GEMINI_API_KEY
3. **降级测试**：在部署前测试AI不可用的情况下系统是否正常工作
4. **用户提示**：确保用户了解AI功能的可选性，以及在AI不可用时如何使用手动功能

## 核心功能保证

即使在AI服务完全不可用的情况下，以下核心功能仍然可以正常使用：

- ✅ 项目管理和查看
- ✅ BOM清单创建和导出
- ✅ 材料库管理
- ✅ 数字化资产上传和管理
- ✅ 用户权限管理
- ✅ 商机线索管理

## 错误提示信息

系统会在以下情况下显示相应的提示信息：

1. **AI服务不可用**："AI服务暂时不可用，请手动输入HS编码"或"AI服务暂时不可用，请使用手动添加项目"
2. **AI生成失败**："AI未能生成有效HS编码，请手动输入"或"AI未能生成有效项目信息，请使用手动添加"
3. **网络错误**："网络错误，请稍后重试"

这些提示信息会以友好的方式显示给用户，确保用户知道如何继续操作。

## 最佳实践

1. **优先使用手动功能**：对于关键操作，建议用户熟悉手动操作流程
2. **定期测试**：定期测试AI服务的可用性，确保系统在各种情况下都能正常工作
3. **用户培训**：为用户提供培训，使他们了解AI功能和手动替代方案
4. **监控和日志**：监控AI服务的使用情况，及时发现和解决问题

通过以上措施，我们确保了即使在AI服务不可用的情况下，系统的核心功能仍然可以正常运行，为用户提供稳定可靠的体验。
